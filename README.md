# 3. 💡 LC-Eye 프로젝트
> **기간:** 2025.11.05 ~ 2025.11.27 (3주)  
> **기술스택:** Spring Boot · React · Flutter  
> **역할:** 팀장 / LCI 계산 알고리즘 구현 / 일정 및 노션 관리  

**LC-Eye**는 경량형 LCI(Life Cycle Inventory) 계산 시스템으로,  
제품·공정 단위의 투입물·산출물 데이터를 기반으로 환경부하를 정량화하는 툴입니다.

**핵심 구현**
- 단위공정별 투입·산출 총량 계산 및 단위 변환 반영
- 키워드 유사도(Text Similarity)를 활용한 자동 매핑 알고리즘 구축
- 기업별 데이터 독립성 및 회원 계층별 접근 권한 설정  

**성과**
- 중복 입력 최소화 및 매칭 정확도 향상(자동 매핑 약 90% 유사도 기준)
- 3주 내 LCI 계산 기능 및 데이터베이스 연동 성공적 구현  

> 🔗 [Notion 바로가기](https://lceye.notion.site/LC-Eye-2a2094d4983480369aa4fe1a6163688f)

# 4. 🗺️ K-Tour 프로젝트
> **기간:** 2025.09.26 ~ 2025.10.31 (4주)  
> **기술스택:** Spring Boot · React · MyBatis · Firebase  
> **역할:** 팀장 / CRUD · 페이지 처리 · 검색 인터페이스 구현  

**K-Tour**는 지자체별 관광정보를 제공하는 SaaS 기반 관광지도 서비스입니다.  
공공데이터(TourAPI)를 정규화하여 **지자체별 맞춤형 도메인·DB 구조**로 운영되는 것이 특징입니다.

**핵심 구현**
- 멀티 도메인 및 DB 기반 SaaS 구조 설계  
- 공공데이터 기반 관광지 정보 정규화 및 마커 커스텀  
- 공통 인터페이스 기반의 페이지·검색 처리 시스템 구축  

**성과**
- 중복 SQL 최소화 및 도메인 확장성 확보  
- Firebase 푸시 알림 서비스로 사용자 경험 강화

## 5. 🎭 공연 5분전 프로젝트
> **기간:** 2025.08.22 ~ 2025.09.12 (3주)  
> **기술스택:** Spring Boot · JSP · MyBatis  
> **역할:** 조장 / 프론트 공통 레이아웃 및 CRUD 인터페이스 구축  

**공연 5분전**은 공연 및 출장 프로젝트 관리 시스템으로,  
프로젝트 단위로 **업무, 역할, 템플릿, 근무 시간, 알림, 채팅** 기능을 통합 구현했습니다.

**핵심 구현**
- 역할 및 체크리스트 템플릿 관리  
- 프로젝트 생성/관리, Excel·PDF 출력  
- 1:1 및 그룹 채팅 기능 구현  
- 공통 CRUD 인터페이스(`CommonRepository`, `AbstractService`) 도입  

**성과**
- 코드 중복 40% 이상 감소, 유지보수성 개선  
- 엑셀 NullPointerException 트러블슈팅 및 안정적 출력 로직 구축
